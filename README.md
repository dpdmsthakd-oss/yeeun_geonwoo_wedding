## 🛠 관리 가이드

### 1. 사진 수정
현재 메인 사진 파일명은 `main.jpg.jpg`입니다. 사진을 바꾸고 싶다면 같은 이름으로 덮어씌우거나, `index.html`에서 파일명을 수정하십시오.

### 2. 계좌번호 및 문구 수정
실제 계좌번호나 하객분들께 전할 문구를 수정하려면 `index.html` 파일의 중간 부분을 수정하시면 됩니다.

### 3. 기능 안내
- **물방울 애니메이션:** 페이지 로드 시 하트로 변하며 낙하하는 효과
- **인터랙션:** 봉투 클릭 시 편지지 슬라이드 업 효과
- **반응형:** 모바일 화면에 최적화된 레이아웃
