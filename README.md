# 🍂 건우 ♥ 예은 결혼합니다

> **2026년 10월 3일 토요일 오후 12:30** > **한국기독교연합회관 3층**

이 저장소는 저희의 소중한 시작을 알리는 **모바일 청첩장** 소스코드입니다. 가을의 정취를 담은 따뜻한 디자인과 우아한 선율로 하객 여러분을 맞이합니다.

---

## ✨ 주요 기능 및 특징

- **🍂 가을 테마 디자인**: 머스터드 골드와 메이플 브라운 톤을 사용하여 10월의 분위기를 연출했습니다.
- **🍁 애니메이션 효과**: 화면 위로 우아하게 흩날리는 단풍잎과 은행잎 효과를 적용했습니다.
- **📱 모바일 최적화**: 다양한 기기에서도 사진이 치우치지 않고 중앙에 정렬되도록 설계되었습니다.
- **🎵 배경 음악(BGM)**: 우측 하단 버튼으로 제어가 가능합니다.
- **🖼️ 갤러리 슬라이더**: 3x3 그리드 형태의 갤러리와 클릭 시 전체 화면으로 볼 수 있는 슬라이더 기능을 포함합니다.
- **📅 스마트 달력**: 2026년 10월 예식일을 빨간 하트로 표시한 직관적인 달력을 제공합니다.
- **🗺️ 오시는 길 안내**: 약도 이미지와 함께 네이버/카카오 지도 연동 버튼, 상세 교통 정보를 제공합니다.
- **💳 마음 전하실 곳**: 계좌번호 복사 기능을 지원하며, 모바일 화면에서도 한 줄로 정렬되어 깔끔하게 보입니다.

---

## 🛠️ 기술 스택

- **HTML5 / CSS3**: 구조 설계 및 레이아웃, 종이 질감(Texture) 구현
- **JavaScript (Vanilla JS)**: D-DAY 카운트다운, 모달 제어, 복사 기능
- **Swiper.js**: 갤러리 슬라이더 애니메이션
- **AOS (Animate On Scroll)**: 스크롤에 따른 부드러운 페이드 효과 구현
- **GitHub Pages**: 웹 호스팅

---

## 📂 파일 구성 안내

- `index.html`: 청첩장의 메인 구조와 스타일, 로직이 포함된 핵심 파일
- `main.jpg`: 메인 표지 및 카톡 썸네일용 사진
- `1.jpg ~ 9.jpg`: 갤러리에 표시될 웨딩 사진들
- `map.jpg`: 예식장 약도 이미지
- `bgm.mp3`: 배경 음악 파일
